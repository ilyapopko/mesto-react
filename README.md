# Проект: "Место-реакт".
### Обзор
Практическая работа 10,11 для Яндекс.Практикума. Проект посвящен обмену фотографиями в интернете.
## Языки разработки
* HTML
* CSS
* JavaScript

## При разработке использованы:
* Нативная верстка
* Элементы flexbox
* Элементы grid
* Фреймворк React
